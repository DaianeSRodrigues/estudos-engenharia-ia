# Jornada: Engenheiro de IA Júnior
Este repositório documenta a minha evolução técnica rumo ao cargo de Engenheiro de IA, seguindo um roteiro intensivo de 3 meses focado em aplicações reais, escalabilidade e boas práticas de engenharia.

## Objetivo do Repositório
O objetivo é construir uma base sólida em integração de LLMs, arquitetura de sistemas baseados em agentes e orquestração de dados para IA.
